# Bangla Numbers 

Bangla數通常會用'kuti' (10000000), 'lakh' (100000), 'hajar' (1000), 'shata' (100)這幾個字來把一個數值轉換成文字。你的任務就是寫一個程式來作這件事。 

輸入檔包含多筆測試資料，每筆測試資料都是一個不超過999999999999999的數字。 

對每一筆測試資料輸出一列轉換後的結果，每一列的開頭必須是一個佔四個字元的case number。
